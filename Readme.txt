Este conversor transfere as unidades de graus, minuto, segundo para coordenadas decimais.
Utiliza transformer() para aplicá-las no EPSG: 31980.

## Como executar

**Windows (com Python instalado):**

Dê dois cliques no arquivo `run.bat` (não incluso no repositório).
Ou, pelo terminal:
python main.py

Deve ser adaptado para realizar transformações para outros EPSG, a critério do usuário.
